# HR Analytics Dashboard (Power BI)

## 📊 Overview
This project analyzes HR data to understand employee attrition, workforce trends, and salary distribution. The dashboard provides interactive visualizations to support HR decision-making.

## 📁 Data Source
- HR dataset in CSV format
- Data imported into SQL table for processing and analysis

## 🛠 Tools & Technologies
- SQL (data import and basic queries)
- Microsoft Power BI (data visualization and dashboard)
- CSV Dataset

## 🔍 Key Analysis
- Employee attrition analysis
- Department-wise employee distribution
- Salary insights
- Gender-based analysis

## ⚙️ Process
1. Imported CSV data into SQL table  
2. Performed basic SQL queries
3. Connected SQL data to Power BI  
4. Built interactive HR Analytics dashboard  


## 📂 Project Files
- HR_Analytics_Project.pbix → Power BI dashboard
- HR_Analytics_Project_Table_Query.sql → SQL script for table creation, data import, and queries
- WA_Fn-UseC_-HR-Employee-Attrition.csv → Dataset used for analysis

## 📌 Conclusion
The dashboard helps identify attrition patterns and workforce trends, enabling better HR decisions.
